The Replenishment Journey V2

Four product lanes. Five emails plus SMS per lane. A 60/90-day split test baked in. Built for RMDY Brands — ready to brief Klaviyo when approved.

4 Product Lanes 5 Emails + 1 SMS per lane 60/90 Day Split Test 3-Lane Status Fork Trigger: Placed Order Auto-exit: on reorder
The V2 Principle

Launch at 60 days. Split-test to 90. Let your customers tell you when they're ready to reorder.

V1 validated that 58-day median reorder cadence. V2 uses that data to run a proper timing test — 50% get the 60-day window, 50% get 90. Emails 1–3 are shared. Only the reorder push diverges. Winner by reorder rate, first read at 90 days post-launch.

06

Klaviyo Build Logic

Four product lanes, one split test, three membership branches. Here's how to build it cleanly.

▶Trigger

Fires on the Placed Order metric. Scope each of the 4 lane flows to the relevant product(s) via trigger filters. Separate flow per lane, or use product-based conditional at entry.

✕Exclusions at Entry

Active Auto-Replenish subscribers excluded — their product ships on its own. Re-entry guard: profile cannot enter if already active in this flow. Suppress by active subscription tag.

⇄60/90 A/B Split

Set a Klaviyo A/B test on the trigger — 50% to arm A (60-day), 50% to arm B (90-day). This is a flow-level split, not an email-level test. Both arms share Emails 1–3. Only the timing of SMS + Emails 4 + 5 differs.

✓Auto-Exit on Reorder

Before every send, a conditional split checks: "Placed Order since flow started." If yes → exit. Nobody gets chased after they convert. This applies to all 5 emails and the SMS.

⇃Email 4 — Membership Fork

Three-way conditional split at Email 4:
Check 1: Active Riversol+ member → Lane A
Check 2: Cancelled or paused Riversol+ → Lane C
Default: Never joined → Lane B

⚙Universal Settings

Smart Sending: On for all emails.
UTMs: On every link.
Support CTA: Route to live chat, not reply-to.
Canadian spelling throughout.
Claims: Appearance-based only — never "reduces," always "the appearance of."

🇨🇦Made in Canada Flag

Show "Made in Canada" for CA-locale recipients. Suppress for US recipients via Klaviyo conditional content block on recipient country. Note in the Klaviyo template so the build team doesn't miss it.

🛡Trust Banner

Include the trust banner universal block (saved in Klaviyo as 'trust banner') in Emails 1, 2, 3. Anchors: Dr. Jason Rivers, clinically tested, 60-day money-back, Made in Canada (CA only).
